# Presentacion Reveal.js con estilo de referencia

Esta carpeta incluye una plantilla Reveal.js inspirada en el estilo visual del PDF de referencia (sin copiar contenido tematico).

## Ejecutar rapido

1. Abre `index.html` en el navegador para una vista inmediata.
2. Para mejor compatibilidad (fuentes, plugins y recarga), usa un servidor local.

### Opcion Python

```bash
python -m http.server 8000
```

Luego abre:

http://localhost:8000

## Editar contenido

- Diapositivas: `index.html`
- Tema visual: `assets/theme.css`

## Rasgos del tema implementado

- Fondo grafito oscuro
- Acento verde lima para enfasis
- Titulos con estilo tecnico/monoespaciado
- Tarjetas grises con borde suave
- Layouts divididos en 2 columnas tipo portada y comparativas
- Ajustes responsivos para movil
